GitHub Build Loaded Into Ordinals

This file lives at /build/index.html in the GitHub repo. The GitHub loader fetches it, sends it into the inscribed receiver using postMessage, and the receiver renders it inside the Ordinals viewer.

One receiver inscription Inscribe the receiver once and reuse it for unlimited projects.
Dynamic builds Update the GitHub build and reload without reinscribing the receiver.
Open HTTPS support GitHub Pages, Vercel, Netlify, Cloudflare Pages, and HTTPS domains can use it.
